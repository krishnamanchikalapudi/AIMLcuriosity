# Superviced Learning

## Prerequsite
- Python 3.8 or higher
- PIP 21.1 or higher

## Update pip tool
``````````
python3 -m pip install --upgrade pip
``````````

## Download dependency libraries
``````````
pip3 install -r requirements.txt
``````````

## Compile files
``````````
python3 -m compileall -l ./scripts/
``````````

## Execute files
- python3 scripts/ScikitLinearRegression.py
